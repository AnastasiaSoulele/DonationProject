# Donation Project

A Java web application for managing donations between donators,
beneficiaries, and organizations. Built with JSP & Servlets as
a university Web Programming project.

##  Overview

The platform connects donors with beneficiaries through organizations,
supporting material donations and service requests.

##  Structure

| Package | Description |
|---------|-------------|
| `Admin` | Admin panel & management |
| `User` | User authentication & profiles |
| `Donator` | Donor registration & donations |
| `Beneficiary` | Beneficiary profiles & requests |
| `Organization` | Managing organizations |
| `Material` | Material donation handling |
| `Service` | Service-type donations |
| `Offers` | Available donation offers |
| `Requests` | Donation requests |
| `RequestDonation` | Linking requests to donations |
| `RequestDonationList` | Listing matched donations |
| `Entity` | Core entity classes |
| `Menu` | Navigation & routing |
| `Main` | Application entry point |

##  How to Run

Deploy on **Apache Tomcat**:

1. Import project into NetBeans / Eclipse
2. Configure your database connection
3. Deploy to Tomcat server
4. Access via `http://localhost:8080/DonationProject`

##  Technologies

- JSP (JavaServer Pages)
